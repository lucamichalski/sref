Midjourney Parameter
Seitenverhältnis iBildformat des Outputs. 3:4 / 4:5 für Adventskalender-Türchen (Hochformat). 1:1 für Instagram-Quadrat. 16:9 für Banner oder Hintergründe.
Version iMidjourney Modell-Version. 6.1 ist aktuell die beste Wahl — detailreichste Ergebnisse, versteht komplexe Prompts am besten.
Style Mode iRaw: MJ greift weniger ein — fotorealistischere, direktere Ergebnisse. Empfohlen wenn du genaue Kontrolle willst.

Default: MJ fügt eigenständig künstlerische Interpretation hinzu — oft ausdrucksstärker, aber weniger vorhersehbar.
Quality iRenderzeit und Detailtiefe. 2 = doppelte Renderzeit, maximale Details — ideal für Druckprodukte. 1 = Standard. 0.5 = schnell für Tests.
Stylize (0–1000) iWie stark MJ seinen eigenen ästhetischen Stil einbringt. 0 = sehr prompt-treu, wenig Eigeninterpretation. 1000 = maximale MJ-Ästhetik, kann vom Prompt abweichen. Empfehlung: 100–300 für Adventskalender.
Chaos (0–100) iZufälligkeit der Ergebnisse. 0 = konsistente, vorhersehbare Bilder. 100 = maximale Variation zwischen den 4 Grid-Optionen. Für Konsistenz bei 0 lassen.
Stop (10–100) iBricht den Render frühzeitig ab. 100 = vollständiges Bild (normal). 50 = halbfertiges, skizzenhafteres Ergebnis. Nützlich für bewusst unscharfe oder traumhafte Ästhetiken.
Weird (0–3000) iFügt ungewöhnliche, experimentelle Elemente hinzu. 0 = normal. Werte ab 500 produzieren surreale, unerwartete Ergebnisse. Für Standard-Adventskalender bei 0 lassen.
Seed iFixiert den Zufallsgenerator. Gleicher Seed + gleicher Prompt = sehr ähnliche Ergebnisse. Nützlich um Variationen eines bestimmten Bilds zu erzeugen oder Ergebnisse zu reproduzieren.
Tile iErzeugt ein nahtlos kachelbares Muster — das Bild kann ohne sichtbare Kanten wiederholt werden. Ideal für Verpackungsdesign, Stoff- oder Tapetenmuster.
Repeat (1–40) iGeneriert mehrere Variationen gleichzeitig. 4 = 4 verschiedene Versionen in einem Job. Praktisch um schnell Alternativen zu vergleichen. Nur in Fast/Relax Mode verfügbar.
Personalize (--p) iPersönlicher MJ-Stil-Code. Wenn du in MJ unter Einstellungen → Personalization einen eigenen Stil trainiert hast, trägst du hier deinen Code ein — MJ gewichtet Ergebnisse dann nach deinen persönlichen Ästhetik-Präferenzen.
Negative Prompt (--no) iElemente die nicht im Bild erscheinen sollen. Komma-getrennt. Beispiel: snow, people, text. Hinweis: Text, Wasserzeichen und Logos werden automatisch ausgeschlossen — hier nur zusätzliche Ausschlüsse eintragen.
Wähle eine Zielgruppe — branchenspezifische Klischee-Tokens werden automatisch in den Prompt eingeflochten.
